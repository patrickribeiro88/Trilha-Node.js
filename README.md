# Trilha Node.js

> Projeto acadêmico desenvolvido no curso de Análise e Desenvolvimento de Sistemas da **UNISENAI**.

Repositório com uma trilha de estudos de **Node.js**, organizada em módulos progressivos — do básico até tópicos mais avançados como autenticação, APIs RESTful e integração com banco de dados MySQL.

## 🎓 Sobre o projeto

Este repositório foi criado como parte das atividades acadêmicas da disciplina de desenvolvimento back-end da UNISENAI, com o objetivo de consolidar o aprendizado de Node.js através da prática, seguindo uma progressão lógica de conteúdos.

## 🚀 Tecnologias

- **JavaScript**
- **Node.js**
- **EJS** (templating)
- **MySQL**
- **HTML/CSS**

## 📚 Estrutura da trilha

| Módulo | Descrição |
|---|---|
| `01 - introducao-ao-node` | Primeiros passos com Node.js: ambiente de execução, módulos e conceitos iniciais. |
| `02 - fundamentos` | Fundamentos da linguagem e do runtime Node.js. |
| `03 - integracao-com-mysql` | Conexão e manipulação de dados usando MySQL. |
| `04 - desenvolvimento-web` | Construção de aplicações web com Node.js. |
| `05 - templating-e-interface-com-servidor` | Renderização de views no servidor com engines de template (EJS). |
| `06 - autenticacao-e-seguranca` | Autenticação de usuários e boas práticas de segurança. |
| `07 - aplicacoes-avancadas` | Tópicos e recursos mais avançados no desenvolvimento com Node.js. |
| `08 - apis-restful-e-mysql` | Criação de APIs RESTful integradas ao banco de dados MySQL. |

## ▶️ Como usar

1. Clone o repositório:
```bash
   git clone https://github.com/patrickribeiro88/Trilha-Node.js.git
```
2. Acesse a pasta do módulo desejado:
```bash
   cd "Trilha-Node.js/01 - introducao-ao-node"
```
3. Instale as dependências (quando houver `package.json`):
```bash
   npm install
```
4. Execute o projeto:
```bash
   node index.js
```

> Cada módulo pode ter suas próprias dependências e instruções específicas — consulte a pasta correspondente para mais detalhes.

## 🎯 Objetivo

Consolidar os conhecimentos em Node.js adquiridos ao longo do curso, aplicando na prática os conceitos vistos em aula: da introdução à linguagem até a construção de aplicações web completas com autenticação e APIs.
