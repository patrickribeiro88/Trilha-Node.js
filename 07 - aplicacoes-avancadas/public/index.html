Exercícios de Node.js Avançado

Projeto com exemplos de async/await, worker threads, socket.io, cluster, child process, PM2 e Winston.

  • npm start — servidor HTTP com API
  • npm run start:socket — servidor Socket.io
  • npm run start:cluster — cluster com múltiplos workers
  • npm run start:child — processo filho
  • npm run start:worker — worker thread
  • npm run start:monitor — monitoramento em background
